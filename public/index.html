Jay's Frames Expert API

Welcome to Jay's Frames Expert

This is the API server for the Jay's Frames Expert application, an AI-powered custom framing design assistant.

Available Endpoints:

POST /api/analyze - Analyze artwork and get framing recommendations

POST /api/contact - Submit contact form information

How to Use

The API accepts artwork images for analysis. Send a POST request to /api/analyze with an image file and optional metadata to receive custom framing recommendations.
