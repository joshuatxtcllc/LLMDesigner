Getting Started

Uploading Artwork

To get framing recommendations for your artwork:

  1. Navigate to the home page
  2. Click on "Upload Artwork" in the hero section
  3. Select an image file of your artwork
  4. Fill in the optional details (dimensions, medium, etc.)
  5. Click "Analyze Artwork"
  6. Wait for the AI to generate recommendations

Understanding Recommendations

The AI will provide three framing options:

  • Traditional Option: Classic framing approach
  • Contemporary Option: Modern framing style
  • Budget Option: Cost-effective solution

API Documentation

Available Endpoints

POST /api/analyze

Upload artwork for analysis

Form Data:

  • artwork: (file) Image of the artwork
  • artworkName: (string) Optional name of the artwork
  • medium: (string) Optional medium of the artwork
  • width: (number) Optional width in inches
  • height: (number) Optional height in inches
  • specialNotes: (string) Optional special considerations

POST /api/contact

Submit contact form

JSON Body:

  • name: (string) Full name
  • email: (string) Email address
  • phone: (string) Optional phone number
  • message: (string) Contact message

Training Mode

Training the AI

For administrators who want to improve the AI with their expertise:

  1. Navigate to the Training section
  2. Click "Start Training Session"
  3. Upload an artwork and make framing decisions
  4. Add annotations explaining your choices
  5. Click "Stop Training" when finished
  6. Your design process will be saved to improve future recommendations
